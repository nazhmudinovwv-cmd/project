## 📱 О проекте

**FleetTrack** — это мобильная PWA-платформа для управления автопарком предприятия в реальном времени. Система объединяет водителей, диспетчеров и руководство в единый цифровой рабочий процесс.

### Ключевые возможности

- 🗺️ **GPS-мониторинг** — интерактивная Canvas-карта г. Рудный с отслеживанием транспорта
- 📋 **Наряды в реальном времени** — диспетчер назначает задачи, водители выполняют с фотоподтверждением
- 📊 **Аналитика для директора** — прогресс каждого экипажа, статусы, телеметрия
- 💬 **Рабочий чат** — корпоративный мессенджер для всех участников
- 📷 **Фотоотчёты** — подтверждение выполненных объектов через камеру

## 🚀 Быстрый старт

### Вариант 1 — Открыть файл напрямую

```bash
# Просто скачай и открой в браузере
open index.html
```

### Вариант 2 — Локальный сервер

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .
```

Затем открой `http://localhost:8080`

### Вариант 3 — GitHub Pages (живая ссылка)

1. Зайди в репозиторий → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / `root`
4. Сохрани — сайт появится по адресу `https://ТВО_ИМЯ.github.io/fleettrack/`

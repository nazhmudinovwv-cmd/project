# 👥 Роли и логика системы FleetTrack

## Водитель (Driver)

**Экраны:** Маршрут → Карта → Чат

- Видит только свои назначенные наряды
- Может переключаться между объектами
- Подтверждает выполнение через фотокамеру
- На GPS-карте видит только свой транспорт
- Может пропустить объект с указанием причины

## Диспетчер (Dispatcher)

**Экраны:** Наряды → Карта → Чат

- Создаёт и назначает задачи водителям
- Видит загрузку всех экипажей
- Может переключаться на GPS любого водителя
- Пишет в общий рабочий чат

## Директор (CEO)

**Экраны:** Аналитика → Карта

- Видит аналитику всего автопарка в реальном времени
- Прогресс каждого экипажа в процентах
- GPS-телеметрия любого транспортного средства
- Статусы: "На маршруте" / "Межсменный отдых"

---

## Статусы нарядов

| Статус | Цвет | Описание |
|--------|------|----------|
| `active` | Синий | В работе (текущий объект) |
| `success` | Зелёный | Выполнено с фотоотчётом |
| `skipped` | Красный | Пропущено с причиной |
| `pending` | Серый | Ожидает очереди |
